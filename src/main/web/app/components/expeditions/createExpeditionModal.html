New Expedition

In this system we refer to each dataset as an “Expedition” each of which includes the metadata and sequences uploaded from a single dataset.

We recommend you name your expedition using following format GenusSpecies_markerabbreviation_yourInitials. If your files include data from multiple taxa please replace GenusSpecies with a relevant descriptor such as GenusSpp. or group such as seaurchins

Expedition Code must contain only numbers, letters, or underscores and be 4 to 50 characters long
